# PassGuard

Веб-приложение на Django для безопасной проверки паролей на наличие в публичных утечках данных, генерации сложного пароля и анализа сложности уже существующего.

## Основные функции

1.  **Проверка пароля на утечку**
    *   Пароль хэшируется в SHA-1 на сервере.
    *   Внешнему API отправляются **только первые 5 символов** хэша.
    *   API возвращает список хэшей с таким префиксом.
    *   Полное совпадение ищется локально на сервере.

    **ВАЖНО:** Пароль и хэш целиком **НИКОГДА** не покидают сервер!

2.  **Анализ сложности пароля** *(в разработке)*
3.  **Генератор пароля** *(в разработке)*

## Зависимости и установка

Установите зависимости в активированном виртуальном окружении:

```bash
pip install django
pip install requests
```
## Запуск проекта(локально)
```
python -m venv .venv
source .venv/bin/activate
pip install django requests
python manage.py migrate
python manage.py runserver
```
